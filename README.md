# YouTube Statistics Analysis (2023)

## Project Overview
This project involves an analysis and visualization of data gathered from YouTube in 2023. The goal of this repository is to explore trends, engagement metrics, and channel statistics using a combination of R for data processing/analysis and Power BI for interactive data visualization.

## Files Included

* **`data.csv`**: The raw dataset containing the YouTube statistics used for this analysis.
* **`YouTube Statistics 2023.Rmd`**: The R Markdown script used for data exploration, cleaning, and statistical analysis.
* **`YouTube-Statistics-2023.html` / `YouTube-Statistics-2023.pdf`**: The rendered output reports generated from the R Markdown file.
* **`Youtube Stats 2023.pbix`**: The Power BI file containing the interactive dashboard built from the dataset.
* **`Youtube Stats 2023-powerBI.pdf`**: A static, exported PDF version of the Power BI dashboard.


## Tools Used
* **R & R Markdown:** For data manipulation and statistical reporting.
* **Power BI:** For building an interactive business intelligence dashboard.

## How to View
* **To view the analysis:** You can download the `YouTube-Statistics-2023.pdf` or open the `.html` file in any web browser.
* **To view the interactive dashboard:** Download the `Youtube Stats 2023.pbix` file and open it using Power BI Desktop. Alternatively, you can view the static snapshot in the `Youtube Stats 2023-powerBI.pdf` file.
